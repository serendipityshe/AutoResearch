# Template Writing Logic

Use the provided template for writing logic, not scientific content.

## Reusable Template Moves

- Abstract progression: clinical problem -> current AI limitation -> `We propose` method -> core mechanism -> datasets/validation plan -> no fabricated results.
- Introduction progression: clinical importance -> low contrast and blurred boundaries -> pixel-mask limitation -> why context plus topology is needed -> contributions.
- Related Work grouping: static segmentation, thyroid-region priors, boundary/graph/topology, uncertain-region methods, cross-dataset generalization.
- Study Design realism: datasets, inclusion/exclusion, preprocessing, implementation preset, baselines, metrics, ablations, statistics.
- Planned Results: table shells with `TBD`.
- Risks and Discussion: restrained claims, explicit validation gaps.

## Forbidden Template Reuse

Do not copy the template's scientific method, datasets, citations, title, claims, or result narrative into a new topic unless supported by the design artifacts.
