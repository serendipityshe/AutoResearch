# Section Architecture

## Required Sections

- Abstract
- Introduction
- Related Work and Design Rationale
- Problem Statement
- Method
- Planned Study Design
- Planned Results Presentation
- Expected Contributions
- Risks
- Discussion
- Conclusion
- Bibliography

## Section Requirements

Abstract must avoid internal wording such as `this document`, `manuscript-ready`, or `study design draft`.

Related Work must end with `Why direct transfer is insufficient` to show that the method is not a loose combination of modules.

Problem Statement must define `x`, `M`, `P`, `R_g`, `\hat{M}`, dataset notation, objective, and constraints.

Study Design must include dataset roles, inclusion/exclusion, preprocessing, implementation preset, baselines, metrics, ablation, and statistics.
